# Overview
> A lightweight but powerful source code editor which runs on your desktop and is available for Windows, macOS and Linux. It comes with built-in support for JavaScript, TypeScript and Node.js and has a rich ecosystem of extensions for other languages (such as C++, C#, Java, Python, PHP, Go) and runtimes (such as .NET and Unity). ~ https://code.visualstudio.com/docs

* Free. Built on open source
* Electron based, written in TypeScript, JavaScript, CSS
  * Similar to Atom
* Feature support varies by language
* Vast number extensions available
* Highly configurable
